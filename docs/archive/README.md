# Archive

This directory contains historical plans, early TODOs, and retired workflow
notes that are kept for context only.

Documents here are not current operator or contributor guidance. Prefer the
top-level README and active docs under `docs/` unless you specifically need
project history.

`retired-runbooks/` contains one-off deployment and audit instructions that
refer to historical branches, phases or infrastructure. Do not execute them as
current production runbooks.
