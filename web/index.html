    What do these numbers mean?

    Everything the planner shows is in two units: öre/kWh for prices (1 SEK = 100 öre) and SEK for whole-horizon totals.

    Header line

    • start SoC — battery state of charge right now. The plan starts from this point.
    • expected cost / earnings — total grid spend over the full horizon (e.g. 48h) at the planner's chosen schedule, using real spot prices, grid tariff and VAT. A negative number means the plan expects to earn money (net export over the horizon).
    • replanned … ago — time since the last optimisation pass; the reason it ran is shown next to it.

    Price bars (top of the chart)

    Each bar is one 15-minute slot, stacked as the three pieces of your real consumer price:

    • spot — raw Nord Pool wholesale price. Colours green / slate / red by tercile across the horizon (cheap / mid / expensive). This is the part the planner actually optimises against.
    • grid tariff — fixed transport / network fee from the grid operator (öre/kWh). Same value every slot.
    • VAT — moms applied to spot + grid tariff. Set in Settings → Price.

    Hover any bar for the exact split (spot / grid / VAT) plus PV, load, battery and SoC for that slot.
